Les bases de SASS et SCSS

  • Pour installer SASS on utilera npm :
    npm install -g sass
  • La commande pour compiler du sass en CSS est :
    sass pathToFile.sass pathToFIle.css
  • Si on souhaite que cela se fasse automatiquement à chaque sauvegarde :
    sass --watch pathToFile.sass pathToFile.css
  • Si on souhaite surveiller "watch" tout un dossier /
    sass --watch pathToFolder:pathToFolderCSS
  • On peut minimiser automatiquement le css avec :
    sass --watch pathToFolderSass:pathToFolderCSS --style compressed

SASS vs SCSS

Les commandes et fonctionnalités sont les mêmes entre SASS et SCSS. Les seules différences sont au niveau de l'extension du fichier ".sass" ou ".scss"et de la synthaxe. SCSS s'écrit comme du css.
SASS s'écrit sans accolade ni point virgule. pour indiquer qu'un propriété est dans un selecteur, on utilisera une indentation.
